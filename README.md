# Anio Music

![home][home-screenshot]

## ✨ 特性

- ✅ 使用 Vue 3 + Typescript + Vite + Tailwind CSS 开发
- 🔴 网易云账号登录（手机/邮箱登录）
- 📻 支持私人 FM / 每日推荐歌曲
- 🚫🤝 无任何社交功能
- 🛠 更多特性开发中

## :computer: 配置开发环境

本项目使用 [Binaryify/NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi) 。

运行本项目

```shell
# 安装依赖
yarn

# 创建本地环境变量
cp .env.example .env

# 运行
yarn dev
```
